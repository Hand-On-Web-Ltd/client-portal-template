# Client Portal Template

A clean, single-page client portal for agencies and freelancers. Let your clients check project progress, download files, and send messages — all from one page.

Built with plain HTML, CSS, and JavaScript. No frameworks, no build tools, no backend required.

## Features

- **Login screen** — Demo mode with pre-filled credentials (no real auth)
- **Project list** — See all projects with progress bars and status labels
- **Milestone tracker** — Track key deliverables and deadlines per project
- **File section** — Browse and download project documents
- **Message thread** — Simple message feed per project
- **Responsive** — Works on desktop, tablet, and mobile

## Quick Start

1. Clone the repo
2. Open `index.html` in your browser
3. Click "Login" (demo credentials are pre-filled)

No server needed — everything runs in the browser using `sample-projects.json`.

## Customising

Edit `sample-projects.json` to add your own projects, milestones, and files. Update the colours in `style.css` to match your brand.
